# Luyện Phỏng Vấn Online

> 1280+ câu hỏi phỏng vấn IT có đáp án chi tiết, song ngữ Việt-Anh.

**🔗 Sản phẩm: [luyenphongvan.online](https://luyenphongvan.online)**

**📅 Cập nhật lần cuối: 29/03/2026**

---

## Giới thiệu

Tổng hợp câu hỏi phỏng vấn IT từ cơ bản đến nâng cao, phân loại theo chủ đề:

- HTML / CSS
- JavaScript / TypeScript
- React / Next.js
- Golang
- System Design
- Design Patterns
- Kafka / Redis
- AWS / Cloud
- Networking / OS
- SEO
- Tools & Misc
- Career & Non-tech

## Tính năng

- Song ngữ Việt-Anh (chuyển đổi ngôn ngữ)
- Dark mode / Light mode
- Tìm kiếm & lọc theo chủ đề, độ khó
- Đóng góp câu hỏi mới qua form
- Responsive — mobile friendly

## Tech Stack

- **Next.js 16** + React 19 + TypeScript
- **Tailwind CSS**
- **Vercel Analytics**
- Deploy trên **Vercel**

## Phát triển

```bash
pnpm install
pnpm dev
```

Mở [http://localhost:3000](http://localhost:3000).

## Đóng góp

Bạn có thể đóng góp câu hỏi mới trực tiếp trên trang web hoặc tạo PR trên repo này.
